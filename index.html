⚡ GFS·ANALYTICSGFS Reporting System — Firebase Setup

One-time setup. Connect your Firebase database so all team members share the same live data.

① Create Firebase Project

  1. Go to console.firebase.google.com → click Add Project
  2. Enter a name (e.g. prod-analytics) → Continue → Create Project

② Create Firestore Database

  1. Left sidebar → Build → Firestore Database → Create database
  2. Choose Production mode → pick your region → Enable
  3. Go to Rules tab → replace the rule with: allow read, write: if true; → Publish

③ Paste Config Below

  1. Left sidebar → ⚙ Project Settings
  2. Scroll to Your apps → click </> Web → Register app
  3. Copy the firebaseConfig object → paste below
